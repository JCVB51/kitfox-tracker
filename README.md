# CMPS 3350 Team 5 Kit Fox Tracker

## Project Description

The Kit Fox Tracker is a web application for recording, viewing, updating, and geographically displaying San Joaquin kit fox sighting reports.

## Course

CMPS 3350 - Software Engineering  
Summer 2026  
California State University, Bakersfield

## Project Goal

Build a deployed web application that allows users to submit and view San Joaquin kit fox sighting reports.

## Core Features

- Create a kit fox sighting report
- Store sighting reports in a persistent database
- View submitted reports
- View details for one report
- Update existing reports
- Hide or delete reports if time allows
- Display sighting locations using a map or map-like view
- Deploy the application to the course server
